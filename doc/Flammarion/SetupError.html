Overview

Raised when flammarion cannot find any way to display an engraving. On Linux, Flammarion will first try to launch Electron using the command electron. If that fails, it will try common aliases of Google Chrome. If none of them execute succesfully, it will raise this error. On Windows, it will try to launch Google Chrome from Program Files (x86). If chrome has been installed somewhere else, the user can set the environment variable FLAMMARION_REVELATOR_PATH to point to chrome.exe.
